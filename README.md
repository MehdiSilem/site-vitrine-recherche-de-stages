# Site Vitrine Recherche de Stages

## Description
Ce site web est une vitrine pour la recherche de stages. Il présente les informations sur les stages disponibles, les mentions légales, et un formulaire de postulation.

## Fonctionnalités
- Page d'accueil (index.html)
- Page des mentions légales (mentions-legales.html)
- Page de postulation (postuler.html)
- Styles CSS (style.css)
- Scripts JavaScript (script.js)

## Technologies utilisées
- HTML5
- CSS3
- JavaScript

## Installation
1. Clonez le dépôt :
   ```
   git clone https://github.com/MehdiSilem/site-vitrine-recherche-de-stages.git
   ```
2. Ouvrez `index.html` dans votre navigateur web.

## Utilisation
Naviguez dans les différentes pages pour explorer le contenu du site.
